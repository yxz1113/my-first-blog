My third post

My personal website design is very simple, because I still can't use css, javascripts website editing language to compile very proficiently. But here you can talk about how django software works. We can explain how django works in this way. First, the user requests the page through the browser. The second step is to grab the user's url and match it through the url.py file to find the corresponding view. Then call the function. The methods in the view can access database data through models and return the data to the view. In the last step, you can use context if you need views. The context is passed to the template to generate the html page. The last step is to return the response object to the browser to present the effect to the user.
